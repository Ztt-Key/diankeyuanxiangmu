# 操作指引功能说明

## 功能概述

操作指引功能为用户提供了一个直观的帮助按钮和弹窗，帮助不熟悉3D系统操作的用户快速了解如何使用本系统。

## 功能入口

在页面右上角（复原按钮上方）有一个蓝色的"操作指引"按钮，点击即可打开操作指引弹窗。

## 组件文件

- 组件路径：`src/components/helpGuide.vue`
- 引用位置：`src/views/Home.vue`

## 操作说明内容

### 基础操作

| 操作 | 功能 |
|------|------|
| 鼠标滚轮 | 缩放视角（控制相机远近）|
| 鼠标左键 + 拖动 | 旋转视角（围绕场景旋转）|
| 鼠标右键 + 拖动 | 平移视角（左右上下移动）|

### 快捷操作

| 操作 | 功能 |
|------|------|
| Shift + 鼠标左键 | 切换相机位置/快速定位 |
| 双击 | 快速聚焦到点击位置 |

### 功能模块

| 模块 | 功能 |
|------|------|
| 首页 | 返回园区整体视图，查看全局场景 |
| 操作仿真 | 进入配电室仿真模式，可以点击配电室进行操作训练 |
| 线缆显示 | 查看园区电力线缆分布和走向 |

### 配电室操作

| 操作 | 功能 |
|------|------|
| 点击配电柜 | 查看配电柜详细信息和回路状态 |
| 复原按钮 | 重置当前视角到默认位置 |
| 回路面板 | 点击回路可查看电路详情和设备状态 |

## 关闭方式

- 点击弹窗右上角的 × 按钮
- 点击弹窗外的遮罩区域
- 按键盘 ESC 键

## 样式特点

- 按钮位置：页面右上角固定位置
- 弹窗风格：深色科技风格，与系统整体UI一致
- 动画效果：弹窗有渐入渐出动画
- 响应式：支持滚动查看完整内容
